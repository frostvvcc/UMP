# 昆明地鐵綜合運營管理平台功能頁面清單

## 1. 安全管理
* 目標管理
* 安全指標
* 安全監察
* 行政執法
* 風險管控（四色風險圖）
* 隱患整治（閉環流轉）
* 違章管理
* 職業衛生管理
* 員工傷害記錄
* 客傷管理與理賠
* 地保管理（施工巡查與沉降監測）
* 消防安全巡檢
* 票務稽查
* 應急管理（預案與物資調度）

## 2. 優質管理
* 服務指標管理
* 服務質量考評（暗訪錄入）
* 投訴與建議追蹤（12345對接）
* 乘客滿意度調查
* 車站環境管理（溫濕度監控）
* 特色服務管理（春城之窗與預約服務）
* 大客流應對複盤
* 專項服務保障方案

## 3. 高效管理
* 目標管理
* 指標管理
* 運作管理（日/周/月報）
* 生產管理（交班記錄）
* 運輸管理（調度日誌、乘務排班、正晚點分析）
* 票務管理（收益清算、AFC設備故障、票卡庫存）
* 設備管理（巡檢計劃、資產台賬、故障報修）
* 施工管理（請銷點審批、動火作業）
* 新線管理（籌備與聯調聯試）

## 4. 綠色管理
* 車站牽引能效分析
* 水電氣網能耗台賬
* 能耗異常報警
* 節能指標設定
* 環控系統節能模式調度

* 車輛段污水處理監控
* 廢舊物資環保處置

* 智能安防監測（越界與聚集預警）

* 員工體檢檔案管理
* 警企聯動協調
